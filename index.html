Descrição da Análise

O projeto consiste em uma análise dos indicadores climáticos do Brasil, extraindo dados das fontes originais, é feito o carregamento em ambientes da Google Cloud Plataform, a correta manipulação dos dados com o uso de bibliotecas python e o devido envio dos dados tratados novamente para ambientes da GCP. Com esses dados tratados é feito as análises disponibilizadas neste site.

Fonte dos dados

Os dados de climas são disponibilizados pelo INMET e estão disponíveis no BDMEP. Referem-se a estações de medições automáticas espalhadas por todo o Brasil.
